# Projeto Piloto - Execucao Imediata

Plano objetivo para iniciar o primeiro app da Fabrica Flutter PerfectPro.

## 1) Escopo do piloto

- Tipo: app CRUD offline-first com PDF.
- Objetivo: validar template, fluxo de desenvolvimento e release.
- Resultado esperado: app funcional + pipeline verde + checklist de release aprovado.

## 2) Sequencia de execucao

1. Criar novo app com FVM (Flutter stable).
2. Aplicar estrutura do `docs/fabrica_flutter_template_manifest.md`.
3. Executar `docs/fabrica_flutter_bootstrap_checklist.md`.
4. Registrar decisoes em `docs/arquitetura.md`.
5. Implementar 1 fluxo completo (cadastro + listagem + PDF).
6. Validar qualidade (`format`, `analyze`, `test`).
7. Fechar com `docs/release_checklist.md`.

## 3) Criterios de sucesso do piloto

- Build Android release sem erro.
- Persistencia local funcionando.
- PDF com caracteres especiais validado.
- Pipeline CI/CD executando.
- Documentacao de arquitetura e changelog atualizada.

## 4) Registro final padrao

- O que foi feito (1 linha)
- Status: OK / ERRO
- Proximo passo sugerido
